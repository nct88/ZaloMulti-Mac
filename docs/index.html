FAQ

Câu hỏi thường gặp

Những thắc mắc phổ biến khi sử dụng Zalỏ macOS.

Zalỏ macOS có can thiệp vào Zalo gốc không?

Không. Zalỏ macOS tạo bản sao Zalo.app riêng biệt (APFS clone), đổi Bundle ID và ký lại. Zalo gốc trong /Applications không bị sửa đổi.

Cần cài Zalo Desktop trước không?

Có. Bạn cần cài sẵn Zalo Desktop chính thức từ App Store hoặc zalo.me. Zalỏ macOS sẽ sao chép từ bản Zalo đó.

Hỗ trợ phiên bản macOS nào?

macOS 14 (Sonoma) trở lên. Hỗ trợ cả Apple Silicon (M1/M2/M3/M4) và Intel.

Dữ liệu clone được lưu ở đâu?

Tại ~/Library/Application Support/ZaloMulti/. Mỗi clone có thư mục riêng, dữ liệu hoàn toàn biệt lập.

Tại sao đồng bộ tin nhắn lần đầu lâu?

Khi đăng nhập clone lần đầu, Zalo coi đó là thiết bị mới nên cần thời gian đồng bộ. Hãy đợi 5–10 phút, không tắt Zalo giữa chừng.
