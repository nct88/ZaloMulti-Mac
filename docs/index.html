Changelog

Có gì mới trong v2.0

Bản cập nhật lớn với trọng tâm bảo mật và hiệu năng.

Bảo mật Mã hóa AES-256-GCM cho toàn bộ dữ liệu nhạy cảm — API endpoints, config, thông tin người dùng.
Bảo mật Anti-tamper: phát hiện công cụ phân tích, kiểm tra toàn vẹn binary, chống injection.
Hiệu năng Thay thế process spawning (pgrep) bằng kernel syscall — giảm 95% CPU khi chạy nhiều nick.
Hiệu năng Notification monitoring chuyển sang kqueue (event-driven) — 0% CPU khi idle.
Hiệu năng Thread-safe avatar cache (NSCache) — giảm memory leak khi chạy nhiều tài khoản.
Mới Cập nhật tự động trong app — tải, cài đặt và khởi động lại chỉ với một nút nhấn.
Sửa lỗi Hotfix bảo mật, cải thiện hiệu năng hệ thống.

FAQ

Câu hỏi thường gặp

Những thắc mắc phổ biến khi sử dụng Zalỏ macOS.

Zalỏ macOS có can thiệp vào Zalo gốc không?

Không. Zalỏ macOS tạo bản sao Zalo.app riêng biệt (APFS clone), đổi Bundle ID và ký lại. Zalo gốc trong /Applications không bị sửa đổi.

Cần cài Zalo Desktop trước không?

Có. Bạn cần cài sẵn Zalo Desktop chính thức từ App Store hoặc zalo.me. Zalỏ macOS sẽ sao chép từ bản Zalo đó.

Hỗ trợ phiên bản macOS nào?

macOS 14 (Sonoma) trở lên. Hỗ trợ cả Apple Silicon (M1/M2/M3/M4) và Intel.

Dữ liệu clone được lưu ở đâu?

Tại ~/Library/Application Support/ZaloMulti/. Mỗi clone có thư mục riêng, dữ liệu hoàn toàn biệt lập.

Cập nhật tự động hoạt động thế nào?

Khi có bản mới, app sẽ hiện thông báo. Nhấn "Cập nhật ngay" để tải và cài đặt tự động — giống Telegram, không cần mở trình duyệt.

Tại sao đồng bộ tin nhắn lần đầu lâu?

Khi đăng nhập clone lần đầu, Zalo coi đó là thiết bị mới nên cần thời gian đồng bộ. Hãy đợi 5–10 phút, không tắt Zalo giữa chừng.
