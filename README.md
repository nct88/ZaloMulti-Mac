# Zalo Multi macOS

Zalo Multi macOS là công cụ quản lý nhiều tài khoản Zalo cùng lúc trên nền tảng macOS, giúp bạn tách biệt công việc và cá nhân một cách dễ dàng, an toàn và tối ưu tài nguyên.

## Tính năng nổi bật
- Chạy nhiều phiên bản Zalo độc lập trên cùng một máy Mac.
- Cơ chế AppleScript đóng mở app an toàn, không gây crash tiến trình.
- Hỗ trợ lưu trữ Keychain riêng biệt để không bị lỗi đăng xuất.
- Giao diện thân thiện, hiện đại, tối ưu cho macOS.

## Hướng dẫn sử dụng (Dành cho người dùng)
Nếu bạn chỉ muốn cài đặt và sử dụng ứng dụng ngay lập tức:
1. Truy cập vào trang [Releases](https://github.com/nct88/ZaloMulti-Mac/releases) của dự án.
2. Tải về tệp cài đặt `ZaloMulti_v...dmg` ở phiên bản mới nhất.
3. Mở tệp DMG vừa tải về và kéo thả biểu tượng **Zalo Multi** vào thư mục **Applications** (Ứng dụng).
4. Mở Zalo Multi từ Launchpad và trải nghiệm.

## Hướng dẫn Cài đặt & Build (Dành cho Lập trình viên)
1. Clone dự án về máy.
2. Mở file `ZaloMulti.xcodeproj` bằng Xcode.
3. Chọn target `ZaloMulti` và nhấn **Run** (Cmd + R) để chạy thử.
4. Chọn **Product > Archive** để Build file `.app`.
